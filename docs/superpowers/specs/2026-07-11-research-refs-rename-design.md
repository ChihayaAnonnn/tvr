# `ref/` 重命名为 `research_refs/` 设计

## 目标

将根目录的本地科研参考材料从 `ref/` 重命名为 `research_refs/`，使目录语义明确为第三方科研项目、论文与本地权重参考，而非项目正式文档。

## 变更范围

- 使用文件系统重命名保留目录内全部内容与忽略状态。
- 将 `.gitignore` 中的 `ref/` 规则替换为 `research_refs/`。
- 更新 `AGENTS.md` 中有关第三方测试和本地参考/权重位置的说明。
- 更新训练、评估、模型默认路径和相应测试，使默认 EVA 实现与权重仍从新目录解析。

## 不变项

- `research_refs/` 继续整体不纳入 Git。
- `docs/` 只保存项目文档；第三方源代码、嵌套 Git 仓库、缓存和本地权重不移动进 `docs/`。
- 不修改现有无关工作树变更。

## 验证

运行针对路径默认值的测试，并搜索受版本控制文件，确认不再存在活动的 `ref/` 路径引用。
